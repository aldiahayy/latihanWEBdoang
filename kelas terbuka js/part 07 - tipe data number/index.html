Tipe Data Number

  1. integer
    2. nilai ini adalah bilangan bulat
  2. float
    3. nilai ini adalah bilangan desimal
  3. bigint
    4. nilai ini adalah bilangan bulat yang nilai nya sangat banyak melebihi 64 bit

cara mengelolah data number

  1. parseInt()
    2. metode ini akan mengonversi bilangan desimal menjadi bilangan bulat dengan cara pembulatan ke bawah
  2. parseFloat()
    3. metode ini akan mengonversi bilangan bulat menjadi bilangan desimal tapi tetap tampil bilangan desimal juga
  3. kegunaan
    4. sebenarnya kedua metode ini berfungsi pada pengonversian string
