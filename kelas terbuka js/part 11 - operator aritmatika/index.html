Operator Aritmatika

  1. * (kali)
    2. ini adalah operator perkalian
  2. / (bagi)
    3. ini adalah operator pembagian
  3. + (tambah)
    4. ini adalah operator penjumlahan
  4. - (kurang)
    5. ini adalah operator pengurangan
  5. % (modulo)
    6. ini adalah operator modulus atau sisa bagi
  6. ++ (increment)
    7. ini adalah operator increment
  7. -- (decrement)
    8. ini adalah operator decrement
  8. ** (power)
    9. ini adalah operator power atau pangkat
