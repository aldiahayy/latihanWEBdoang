String (escaping dan literal)

  1. escaping string
    2. yaitu mengganti tanda kutip dua nya menjadi kutip satu agar tidak terjadi penumpukan atau dengan cara menambahkan tanda backslash sebelum tanda kutip dua nya. escaping karakternya yaitu (\' \" \\ \n \r \t \b \f)
  2. literal string
    3. yaitu penggabungan string tanpa mengubah tipe data yang lain menjadi string juga dengan cara `${namaVariabel}` jika tidak dalam ${} maka itu akan terbaca sebagai string
