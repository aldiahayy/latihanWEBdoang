Operator Logika

Operator Logika akan selalu menghasilkan nilai boolean yaitu true ataupun false

  1. ! (Not)
    2. ini akan membuat data boolean menjadi terbalik yang awal nya true akan menjadi false begitu pula sebaliknya
  2. || (OR)
    3. ini akan membandingkan dua data boolean, jika salah satu true atau semua nya true maka hasil nya akan true, selebihnya false
  3. && (And)
    4. ini akan membandingkan dua data boolean, jika semua nya true maka hasil nya akan true, selebihnya false
