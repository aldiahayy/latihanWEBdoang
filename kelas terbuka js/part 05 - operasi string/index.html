Operasi String

  1. .charAt(index)
    2. ini untuk mengakses karakter di string menggunakan index nya
  2. .concat()
    3. ini untuk menyambungkan string
  3. .indexOf(karakter yang ingin diakses)
    4. ini untuk melihat karakter tersebut ada di index yang mana
  4. .substring(awal, akhir + 1)
    5. ini untuk mengambil karakter awal hingga akhir + 1 atau akhir + 1 hingga awal sesuai dengan index yang di inginkan
  5. .slice(awal, akhir + 1)
    6. sama seperti substring() tapi ketika index nya itu dari akhir + 1 hingga awal maka tidak akan tercetak
  6. .replace(yang ingin diganti, yang jadi pengganti)
    7. ini akan mengganti string sebelum nya menjadi yang baru
  7. .toLowerCase()
    8. ini akan mengubah seluruh nya menjadi huruf kecil semua
  8. .toUpperCase()
    9. ini akan mengubah seluruh nya menjadi huruf besar atau kapital semua
  9. parseInt()
    10. ini untuk mengubah tipe data string menjadi tipe data number yang bilangan bulat
  10. parseFloat()
    11. ini untuk mengubah tipe data string menjadi tipe data number yang bilangan desimal
