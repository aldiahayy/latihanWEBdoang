variabel (let, var, dan const)

  1. let
    2. dia hanya menampilkan data yang hanya dalam kurung kurawal saja
  2. var
    3. dia akan menampilkan data yang terbaru walaupun data nya dalam kurung kurawal
  3. const
    4. dia akan membuat data yang tidak bisa dirubah / diupdate karena nilai nya tetap
